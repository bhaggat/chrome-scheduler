How do I install the extension?
  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm the installation in the popup
  4. Click the extension icon in your browser toolbar to start using it
How do I schedule a website to open?
  1. Click the extension icon in your browser toolbar
  2. Click the "Add Schedule" button
  3. Enter the website URL you want to schedule
  4. Set the date and time for when it should open
  5. Choose the recurrence (daily, weekly, monthly, etc.)
  6. Click "Save" to create your schedule
Can I edit or delete scheduled websites?
Yes! You can edit or delete any scheduled website by clicking the edit or delete buttons next to each scheduled item in the extension popup.
Will my schedules work if I close my browser?
Yes! The extension uses Chrome's alarm system, so your scheduled websites will open even if your browser is closed, as long as Chrome is running in the background.
Is my data safe and private?
Absolutely! All your schedule data is stored locally on your device. We don't collect, store, or share any of your personal information. Your data never leaves your computer.
The extension isn't working. What should I do?
Try these troubleshooting steps:
  1. Make sure Chrome is updated to the latest version
  2. Disable and re-enable the extension
  3. Check if the extension has the necessary permissions
  4. Restart your browser
  5. If the issue persists, contact our support team
Can I schedule multiple websites at the same time?
Yes! You can schedule as many websites as you want, and they can all be set to open at the same time or at different times throughout the day.
Does the extension work on mobile devices?
Currently, the extension is designed for desktop Chrome browsers. It may not work on mobile devices or other browsers.

Extension Not Opening Websites

  • Check if Chrome is running (the extension needs Chrome to be active)
  • Verify that the scheduled time has passed
  • Make sure the website URL is correct and accessible
  • Check if popup blockers are interfering

Extension Not Loading

  • Refresh the extension page
  • Check Chrome's extension management page
  • Ensure you have the latest version installed
  • Try disabling and re-enabling the extension

Schedule Not Saving

  • Make sure all required fields are filled out
  • Check if the date/time is set correctly
  • Verify that Chrome has storage permissions
  • Try clearing the extension's data and starting fresh
